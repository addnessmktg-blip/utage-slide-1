# UTAGE Slide Creation Assistant - クイックスタートガイド

## 概要

**utage-slide-1** は、UTAGEプラットフォーム向けのスライド・ランディングページ作成をサポートするAIアシスタント用のシステムプロンプトです。

OpenAI GPT、Google Gemini Gems、Claude Projectsなど、様々なAIプラットフォームで利用できます。

---

## クイックスタート（3ステップ）

### Step 1: システムプロンプトをコピー

`README.md` を開き、システムプロンプト部分をコピーします。

```
# あなたは～ から 最後まで全体をコピー
```

### Step 2: AIプラットフォームに設定

お使いのAIプラットフォームにシステムプロンプトを設定します：

| プラットフォーム | 設定場所 |
|-----------------|---------|
| **ChatGPT** | 「GPTを作成」→「Instructions」欄に貼り付け |
| **Gemini Gems** | 「Gem を作成」→「指示」欄に貼り付け |
| **Claude Projects** | 「プロジェクト設定」→「カスタム指示」に貼り付け |

### Step 3: 使用開始

設定完了後、アシスタントに話しかけてスライド作成を開始できます。

```
例: 「AIセミナーのスライドを作成したい」
```

---

## 対話フロー（6ステップ）

アシスタントは以下の流れで進行します：

| STEP | 内容 | 説明 |
|------|------|------|
| **1** | コンテンツ目的確認 | スライド？LP？ニュース記事？ |
| **2** | ターゲット確認 | どんな人向けの内容か |
| **2.5** | テーマ・コンセプト | 具体的なテーマの深掘り |
| **3** | デザイン・トーン | デザインの方向性を決定 |
| **4** | 構成確認 | コンテンツの構造を設計 |
| **5** | ボリューム確認 | ページ数・分量の確定 |
| **6** | メディア確認 | 画像・動画素材の確認 |

## デザインテンプレート

4つのテンプレートから選択可能：

### 1. Neural Interface（ニューラルインターフェース）
- **スタイル**: ダーク × テクノロジー
- **用途**: AI/IT講座、先進的なコース

### 2. Apple Style（アップルスタイル）
- **スタイル**: ホワイト × ミニマル
- **用途**: 製品紹介、高級感のある訴求

### 3. Corporate（コーポレート）
- **スタイル**: プロフェッショナル
- **用途**: B2B、ビジネスコンテンツ

### 4. Pop（ポップ）
- **スタイル**: 明るい × フレンドリー
- **用途**: B2C、初心者向けコンテンツ

## 重要な注意事項

### UTAGEプラットフォームの制約

| ルール | 詳細 |
|--------|------|
| ❌ **カスタムCSS禁止** | エディタが壊れる可能性があります |
| ❌ **`<script>`タグ禁止** | HTML内でのスクリプトはセキュリティでブロックされます |
| ✅ **カスタムHTML使用** | HTML + CSS（`<style>`タグ）を記述 |
| ✅ **カスタムJS使用** | JavaScriptはカスタムJS欄に記述 |

### コード仕様

- **JavaScript**: ES5（互換性のため）
- **レスポンシブ対応**: 768px（タブレット）、480px（モバイル）
- **アニメーション**: fadeIn, pulse などサポート

## よくある質問

### Q: インストールは必要ですか？
**A:** いいえ、必要ありません。このプロジェクトはシステムプロンプト（テキスト）のみで構成されています。コピー＆ペーストするだけで使用できます。

### Q: どのAIプラットフォームで使えますか？
**A:** システムプロンプト（カスタム指示）をサポートする任意のAIプラットフォームで使用できます。ChatGPT、Gemini Gems、Claude Projectsで動作確認済みです。

### Q: 生成されるコードはそのまま使えますか？
**A:** はい。UTAGEプラットフォームの仕様に合わせて最適化されたHTML/CSS/JavaScriptが生成されます。
