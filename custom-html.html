01

AIは便利。

企画のアイデア出し、文章作成、リサーチ...
なんでも手伝ってくれる。

でも、落とし穴がある。

TRAP

02

AIは、なんでも肯定する

あなたが何を言っても、基本的に「いいですね！」と返ってくる。
これが、AIの本質。

試してみてください。
どんな企画を見せても、AIは褒めてくれます。

EXAMPLE

例えば、こう聞くと...

You
この企画、どう思う？褒めて！
AI
素晴らしい企画ですね！ターゲット設定も明確で、市場のニーズをしっかり捉えています。特に、競合との差別化ポイントが明確で、実現可能性も高いと思います！

⚠️これ、本当に良い企画かどうかは分からない。
AIは「褒めて」と言われたから褒めただけ。

03

「じゃあ、批判的に考えてって
言えばいいんじゃない？」

...と思いますよね。
実際、多くの人がこれを試します。

EXAMPLE

「批判的に考えて」と言うと...

You
この企画、批判的な視点で見てくれる？
AI
分かりました。批判的に見ますね。ターゲット層がやや広いかもしれません。また、競合との差別化ポイントをもう少し明確にすると良いでしょう。予算配分についても再検討の余地があるかもしれません。

一見、ちゃんと批判してくれてるように見える...

でも、これもイエスマン。

なぜなら...

WHY?

その理由

「批判的に考えて」という
指示に従っただけ

AIは、あなたの指示に忠実に応える。
「批判して」と言われれば批判する。

でもそれは、あなたが想定した範囲内の批判でしかない。
あなたが気づいていない視点からのツッコミは入らない。

POINT

AIは指示に従う。

YOU

指示を出す

→

AI

忠実に実行

→

OUTPUT

指示の範囲内

優秀すぎるがゆえに、
あなたの指示の「外側」には出てこない。

04

つまり、こういうこと

✗ AIができないこと

あなたが想定していない視点から
自発的にツッコミを入れる

→ 指示されていないことはしない

◯ AIができること

あなたが指示したことに対して
忠実に、完璧に答える

→ 指示されたことは完璧にこなす

AIは優秀な部下のようなもの。

言われたことは完璧にやる。
でも、言われてないことは指摘しない。

上司（あなた）の指示を疑わない。
それが「良い部下」だから。

「いいね」って
言われると嬉しい。

認めてもらえると気分がいい。
自分の考えが正しいって思える。

でも、それで本当に良くなる？

自分の穴は、
自分では見えない。

あなたの視野

誰かに「それ、違くない？」と言われて
初めて気づく。

人間は空気を読む。
本音を言わないこともある。

でも、AIは遠慮しない。
頼めば、容赦なく指摘してくれる。

だから、正しく使えば
最強の壁打ち相手になる。

05

じゃあ、どうするか？

答えはシンプル。

「何を検証するか」を
具体的に指示する

「批判的に」という曖昧な指示ではなく、
どんな角度から検証してほしいかを明確に伝える。

具体的な「検証ポイント」を指定することで、
AIは自分の盲点を突いてくれるようになる。

PROMPT 01 / 失敗シナリオ

「この企画が失敗するとしたら、
どんな理由が考えられる？」

→ AIはこう返す

失敗の可能性として、以下が考えられます：
1. ターゲットの購買タイミングと施策時期のズレ
2. 競合の類似キャンペーンとの差別化不足
3. 予算に対するリーチ数の見積もりが甘い可能性...

PROMPT 02 / ターゲット外視点

「ターゲット外の人がこれを見たら、
どう感じると思う？」

→ AIはこう返す

ターゲット外の層から見ると：
・専門用語が多く、何のサービスか分かりにくい
・価格帯が見えず、自分向けか判断できない
・「若者向け」という印象が強く、中高年は敬遠しそう...

PROMPT 03 / 競合視点

「競合がこの企画を見たら、
何て言うと思う？」

→ AIはこう返す

競合の立場から見ると：
・「うちでも同じことできるな」と思われる可能性
・価格競争に持ち込まれるリスク
・差別化ポイントが技術的に模倣しやすい...

PROMPT 04 / 前提を疑う

「この企画の前提が間違っている
可能性はある？」

→ AIはこう返す

以下の前提に疑問があります：
・「ターゲットはSNSを見ている」という前提
・「価格が安ければ売れる」という前提
・「認知さえ取れれば購入につながる」という前提...

PROMPT 05 / 見落とし

「私が見落としている視点が
あるとしたら何？」

→ AIはこう返す

見落としている可能性がある視点：
・法規制やコンプライアンスの観点
・社内リソースの制約（実行可能性）
・季節性や市場トレンドの変化
・既存顧客への影響...

06

そして、一発で終わらない

AIとの対話はキャッチボール。
一度聞いて終わりじゃない。

深掘りを繰り返すことで、
本当に価値のある気づきが出てくる。
