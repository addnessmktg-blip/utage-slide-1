AIはイエスマン

〜 盲点を突かせる技術 〜

AGENDA

今日話すこと

  • 01 AIはイエスマンである
  • 02 なぜそれが問題なのか
  • 03 盲点を突かせる方法
  • 04 今日から使えるプロンプト

01

AIは便利。

企画のアイデア出し、文章作成、リサーチ...
なんでも手伝ってくれる。

でも、落とし穴がある。

02

AIは、なんでも肯定する

あなたが何を言っても、基本的に「いいですね！」と返ってくる。
これが、AIの本質。

EXAMPLE

例えば、こう聞くと...

You
この企画、どう思う？褒めて！
AI
素晴らしい企画ですね！ターゲット設定も明確で、市場のニーズをしっかり捉えています！

⚠️これ、本当に良い企画かどうかは分からない。

03

「じゃあ、批判的に考えてって
言えばいいんじゃない？」

...と思いますよね。

EXAMPLE

「批判的に考えて」と言うと...

You
この企画、批判的な視点で見てくれる？
AI
分かりました。批判的に見ますね。

ターゲット層がやや広いかもしれません。また、競合との差別化ポイントをもう少し明確にすると良いでしょう。

一見、ちゃんと批判してくれてるように見える。

でも、これもイエスマン。

WHY?

なぜか？

「批判的に考えて」という
指示に従っただけ

AIは、あなたの指示に忠実に応える。
「批判して」と言われれば批判する。
でもそれは、あなたが想定した範囲内の批判でしかない。

POINT

AIは指示に従う。
それ以上でも、それ以下でもない。

04

つまり、こういうこと

✗ AIができないこと

あなたが想定していない視点から
自発的にツッコミを入れる

◯ AIができること

あなたが指示したことに対して
忠実に答える

AIは優秀な部下のようなもの。

言われたことは完璧にやる。
でも、言われてないことは指摘しない。

05

じゃあ、どうするか？

「何を検証するか」を
具体的に指示する

「批判的に」ではなく、
どんな角度から検証してほしいかを明確に伝える。

PROMPT 01 / 失敗シナリオ

「この企画が失敗するとしたら、
どんな理由が考えられる？」

→ 成功前提ではなく、失敗の可能性を洗い出させる

PROMPT 02 / ターゲット外視点

「ターゲット外の人がこれを見たら、
どう感じると思う？」

→ 想定外の層からの見え方を確認する

PROMPT 03 / 競合視点

「競合がこの企画を見たら、
何て言うと思う？」

→ 第三者（競合）の目線で弱点を探る

PROMPT 04 / 前提を疑う

「この企画の前提が間違っている
可能性はある？」

→ 土台から疑わせる

PROMPT 05 / 見落とし

「私が見落としている視点が
あるとしたら何？」

→ 自分の盲点を直接聞く

06

そして、一発で終わらない

AIとの対話はキャッチボール。
一度聞いて終わりじゃない。

EXAMPLE

対話を繰り返す

You
この企画が失敗するとしたら？
AI
ターゲットの購買タイミングとズレている可能性があります。
You
それ、もう少し詳しく
AI
この商品カテゴリは年末に需要が集中しますが、企画は春の実施予定ですよね...
You
他にリスクある？

このやり取りを何度も繰り返すことで、

自分では気づけなかった視点が
見えてくる。

SUMMARY

今日のまとめ

1 AIはイエスマン。指示に従うだけ。
2 「何を検証するか」を具体的に指示する。
3 一発で終わらず、対話を繰り返す。

AIはイエスマン。
だから、正しく疑わせろ。

それだけで、AIはあなたの
最強の壁打ち相手になる。
